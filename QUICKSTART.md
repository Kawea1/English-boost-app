# 🚀 快速启动指南

## 立即开始使用英语冲刺通

### 方法一：本地启动（推荐用于开发）

```bash
# 1. 进入项目目录
cd /Users/huangjiawei/english-boost-app

# 2. 使用 Python 启动本地服务器
python3 -m http.server 8000

# 3. 打开浏览器访问
# http://localhost:8000
```

### 方法二：使用 VS Code Live Server

1. 在 VS Code 中打开项目
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"
4. 浏览器自动打开应用

### 方法三：直接打开（功能受限）

直接双击 `index.html` 文件打开（部分功能可能受浏览器限制）

1. **困难单词**：第二天、第三天、第五天各复习一次
2. **一般单词**：3天后复习一次
3. **简单单词**：7天后复习一次，然后标记为"已掌握"

4. **听力训练**：先正常速度，再快速播放挑战
5. **口语练习**：目标评分80+，多次练习直到流利
6. **阅读理解**：先不看词汇表，测试真实水平

## ❓ 遇到问题？

### 语音识别不工作
- 确保使用 Chrome、Edge 等现代浏览器
- 检查麦克风权限是否已授予
- 尝试刷新页面

### 密钥提示已在其他设备激活
- 永久密钥只能在一台设备使用
- 清除浏览器缓存后重试
- 或联系管理员获取新密钥

### 数据丢失
- 不要清除浏览器的 LocalStorage
- 定期导出统计数据备份
- 重新登录后数据会恢复
