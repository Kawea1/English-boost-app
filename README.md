## 项目简介

学术英语精进是一款专业的英语学习Web应用,专注于GRE、TOEFL及学术英语能力提升。本项目采用渐进式Web应用(PWA)架构,支持跨平台使用和离线访问,所有数据存储在本地,保障用户隐私安全。

### 适用人群

- GRE/TOEFL考试备考者
- 学术论文写作需求者
- 英语专业学生及研究人员
- 需要提升学术英语能力的专业人士

## 立即使用

### 在线访问（推荐方式）

访问地址: **[https://kawea1.github.io/English-boost-app](https://kawea1.github.io/English-boost-app)**

无需安装,打开即用。支持桌面浏览器、移动端浏览器全平台访问。

## 核心功能

### 一、词汇学习系统（10000+ 词汇）

**词库覆盖**
- GRE核心词汇
- TOEFL必备词汇
- Academic Word List (AWL) 学术词表
- 高阶学术词汇扩展
- 大学英语六级词汇

**学习功能**
- 智能复习算法:基于艾宾浩斯遗忘曲线理论,科学安排复习间隔
- 完整词汇信息:国际音标、词性标注、中英文释义、例句展示
- 词根词缀解析:帮助理解词汇构成,提升记忆效率
- 发音系统:采用Web Speech API,支持美式/英式英语真人发音
- 学习进度追踪:记录学习历史,统计掌握程度

### 二、听力训练模块（50+ 材料）

**训练内容**
- 学术讲座听力材料
- GRE/TOEFL听力真题风格材料
- 多主题覆盖:自然科学、社会科学、艺术人文

**训练方式**
- 填空式听写练习
- 可调节播放速度 (0.5x - 2.0x)
- 即时答案反馈
- 错误统计分析

### 三、口语训练系统（100+ 句型）

**训练特色**
- 语音识别技术:实时转换语音为文字
- 智能评分系统:0-100分量化评估发音准确度
- 对比播放功能:支持原音与录音对比
- 多场景覆盖:学术演讲、商务交流、日常对话

### 四、阅读理解训练（100+ 文章）

**阅读材料**
- GRE/TOEFL阅读真题风格
- 学术期刊文章节选
- 多领域主题文章

**训练功能**
- 理解题练习
- 核心词汇标注
- 阅读时间统计
- 正确率追踪

### 五、写作练习模块

**支持题型**
- TOEFL独立写作/综合写作
- GRE Issue/Argument写作
- IELTS Task 1/Task 2
- 考研英语作文
- 大学英语六级作文

**辅助功能**
- 限时写作模式
- 词数统计
- 范文参考
- 常用学术词汇提示

### 六、智能复习系统

**复习策略**
- 基于艾宾浩斯遗忘曲线的7个科学复习节点
- 自动识别需复习词汇
- 学习数据可视化统计
- 本地数据自动保存

## 本地开发

### 环境要求

- 现代浏览器（Chrome 90+, Firefox 88+, Safari 14+, Edge 90+）
- 本地Web服务器（推荐使用以下任一方式）

### 启动开发服务器

**使用Python（推荐）**
```bash
# Python 3.x
python -m http.server 8080

# Python 2.x
python -m SimpleHTTPServer 8080
```

**使用Node.js**
```bash
# 使用serve包
npx serve . -p 8080

# 或安装http-server
npm install -g http-server
http-server -p 8080
```

**使用VS Code**
- 安装 Live Server 扩展
- 右键 `index.html` 选择 "Open with Live Server"

启动后访问: `http://localhost:8080`

### 项目结构

└── www/                    # 生产环境资源

## 技术架构

### 前端技术

| 技术栈 | 版本/规范 | 用途说明 |
|:---|:---:|:---|
| **HTML5** | - | 页面结构,语义化标签 |
| **CSS3** | - | 样式设计,响应式布局,动画效果 |
| **JavaScript** | ES6+ | 核心业务逻辑 |
| **Web Speech API** | W3C标准 | 语音合成(TTS)与语音识别(STT) |
| **LocalStorage API** | W3C标准 | 本地数据持久化存储 |
| **Canvas API** | HTML5 | 数据可视化图表绘制 |

### PWA技术

| 技术 | 用途 |
|:---|:---|
| **Service Worker** | 离线访问、资源缓存管理 |
| **Web App Manifest** | 添加到主屏幕、启动配置 |
| **Cache API** | 静态资源缓存策略 |

### 核心算法

- **艾宾浩斯遗忘曲线算法**: 智能复习系统的理论基础
- **字符串相似度算法**: 口语评分的核心算法
- **间隔重复算法(Spaced Repetition)**: 优化词汇记忆效果

### 浏览器兼容性

| 浏览器 | 最低版本 | 备注 |
|:---|:---:|:---|
| Chrome/Edge | 90+ | 完全支持 |
| Firefox | 88+ | 完全支持 |
| Safari | 14+ | 完全支持,iOS 14+ |
| Opera | 76+ | 完全支持 |

## 数据隐私

### 本地存储策略

本应用采用**完全本地化存储**策略:
- 所有学习数据存储在用户浏览器LocalStorage中
- 不收集、不上传任何个人信息至服务器
- 用户可随时导出或清除数据
- 数据仅在本地设备上处理

### 权限使用

- **麦克风权限**: 仅用于口语练习的语音识别,数据实时处理后立即释放
- **通知权限**: 可选,用于复习提醒功能
- **存储权限**: 用于保存学习进度和设置
