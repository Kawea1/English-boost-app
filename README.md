# 英语冲刺通 (English Boost App)

一款全功能的英语学习 PWA 应用，支持 GRE、托福、六级等考试的备考训练。

## 🌟 功能特性

### 📖 核心词汇模块
- **10000+ 高频词汇**：涵盖 GRE、托福、六级核心词汇
- **智能记忆系统**：基于艾宾浩斯遗忘曲线的复习提醒
- **三级难度评分**：困难/一般/简单，自适应调整复习间隔
- **完整释义**：中英文双语释义、音标、例句
- **真人发音**：支持美式英语发音
- **字典集成**：自动查询详细词典释义

### 🎧 精听训练模块
- **互动式听写练习**：填空题形式，即时反馈
- **可调节语速**：0.5x - 2.0x 播放速度
- **真人发音**：TTS 语音合成
- **答案解析**：完整句子对照
- **进度追踪**：记录练习次数

### 🎤 口语跟读模块
- **语音识别**：使用 Web Speech API
- **实时评分**：基于相似度算法的智能评分（0-100分）
- **发音对比**：听原音、录自己、对比结果
- **个性化反馈**：根据得分给出建议
- **多样化句子**：涵盖不同话题和难度

### 📰 阅读精读模块
- **100+ 精选文章**：GRE、托福、学术英语文章
- **理解题训练**：每篇配有多道选择题
- **核心词汇标注**：文章配套重点词汇表
- **即时评分**：答题后立即显示成绩和解析
- **分类浏览**：按类别、难度、话题筛选

### 🔄 复习巩固模块
- **艾宾浩斯记忆法**：科学的复习间隔提醒
- **学习统计**：已学单词、待复习、已掌握数据展示
- **复习计划**：7个时间节点的复习安排
- **智能推荐**：根据遗忘曲线推荐复习内容

### 🌐 学习资源模块
- **官方真题支持**：可导入并解析 PDF 真题
- **在线资源**：TED演讲、学术文章、音频资源链接
- **双资源类型**：本地解析真题 + 在线学习网站
- **一键访问**：直接跳转原文链接

## 🔧 使用说明

### 基本使用

1. **登录**：输入激活密钥（如 `SHAO1`）
2. **选择模块**：点击主页的模块卡片
3. **开始学习**：按照模块指引进行学习
4. **查看统计**：底部导航栏 → 统计

### 词汇学习流程

1. 查看单词
2. 点击"🔊 发音"听读音
3. 点击"显示释义"查看意思
4. 选择难度评分（困难/一般/简单）
5. 系统自动记录并安排复习

### 听力练习流程

1. 选择练习题目
2. 点击播放按钮听音频
3. 在空白处填写答案
4. 点击"检查答案"查看结果
5. 重复练习直到完全掌握

### 口语练习流程

1. 点击"🔊 听原音"听标准发音
2. 点击"🎙️ 点击录音"开始录音
3. 朗读句子后再次点击停止
4. 查看识别结果和相似度评分
5. 根据反馈改进发音

### 阅读练习流程

1. 从列表选择文章
2. 阅读文章和核心词汇
3. 回答下方的理解题
4. 提交答案查看成绩和解析

## 📚 导入官方真题

详见 [README_OFFICIAL.md](./README_OFFICIAL.md)

简要步骤：
1. 将 PDF 真题放入 `official_inputs/` 目录
2. 运行解析脚本：`python3 scripts/ingest_official.py`
3. 在"学习资源"模块查看解析结果

## 🎯 学习建议

### 高效学习路径

1. **每日词汇**（20-30分钟）
   - 新学25-50个单词
   - 复习昨天标记为"困难"的单词

2. **精听训练**（15-20分钟）
   - 完成2-3个听写练习
   - 重点练习错误的部分

3. **口语跟读**（10-15分钟）
   - 练习5-10个句子
   - 力求评分达到80+

4. **阅读精读**（20-30分钟）
   - 精读1-2篇文章
   - 完成所有理解题

5. **复习巩固**（10分钟）
   - 每天查看复习计划
   - 按时完成复习任务

### 艾宾浩斯复习间隔

- **困难**：1天后复习
- **一般**：3天后复习
- **简单**：7天后复习
- 后续间隔：2天、4天、7天、15天、30天

## 🐛 常见问题

### Q: 语音识别不工作？
A: 请使用 Chrome、Edge 等支持 Web Speech API 的浏览器，并授予麦克风权限。

### Q: 密钥提示"已在其他设备激活"？
A: 永久密钥绑定设备，无法在多设备使用。请联系管理员获取新密钥。

### Q: 如何导出学习数据？
A: 进入"统计"页面，点击"导出统计"按钮即可下载 JSON 格式数据。

### Q: 如何离线使用？
A: 首次访问后，应用会自动缓存资源。断网后仍可正常使用所有功能。
